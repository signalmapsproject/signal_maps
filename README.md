# Signal Maps — Iran Conflict · March 2026

An open-source intelligence project applying the **Exoplanet Method** to the Iran conflict: treating information gaps as data, mapping what is visible, missing, delayed, or rerouted, and stress-testing claims by source environment, timing, and proxy indicators.

Built using a two-model relay — **DeepSeek** for source expansion, non-Western routing, and adversarial stress-testing; **ChatGPT** for synthesis, claim classification, and document production.

## What the Project Does

Signal Maps tracks what is **visible, partial, delayed, or missing** across conflict reporting. It does not treat absence as self-proving. It treats absence as a lead for further investigation.

The homepage is organized around:
- **Source visibility**
- **Signal interpretation**
- **Case-based deep dives**
- **Method transparency**
- **AI comparison**
- **Ledger-based tracking**

This project is designed to make the information environment more legible without forcing certainty where the evidence does not support it.

## The Exoplanet Method

Astronomers find planets they cannot see by watching for the shadow they cast — a dip in starlight, a wobble in a star's orbit. The same principle applies to information.

**What does not come through can still leave a trace.**

The Exoplanet Method asks:
- What is visible?
- What seems absent?
- Where does the signal appear instead?
- What rival explanations fit the same gap?
- What proxy indicators change the confidence level?
- Where does inference end and overreach begin?

The method applies to:
- news coverage
- government claims
- platform/search visibility
- AI tool outputs

## Confidence Approach

Signal Maps separates:
- **verified facts**
- **proxy indicators**
- **disputed claims**
- **open hypotheses**
- **narrative framing**

Confidence is not just about who said something. It also depends on:
- source incentives
- corroboration
- timing
- operational constraints
- whether the claim is even structurally verifiable
